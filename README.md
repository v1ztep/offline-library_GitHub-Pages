# Деплой сайта

Из проекта [offline-library](https://github.com/v1ztep/offline-library) необходимо по инструкции сгенерировать сайт локально и скопировать в свой новый репозиторий сайта -> следующие файлы:
* /media
* /pages
* /static
* favicon.ico
